Safety & Transparency

Google Play Protect Handling

Because Z Flow uses Accessibility Services to interact with other apps (to click "Skip Ad"), Play Protect identifies it as a potential risk. To install this developer utility safely, follow these steps:

Mandatory Installation Steps:

  • 1 Open Play Store ➡️ Tap Profile Icon ➡️ Play Protect ➡️ Settings (⚙️ icon).
  • 2 Turn OFF "Scan apps with Play Protect" (This allows the installation to proceed).
  • 3 Install the Z Flow APK.
  • 4 Go to App Info > Z Flow ➡️ Tap 3dots menu ➡️ "Allow restricted settings" to enable Accessibility.

* Note: You can re-enable Play Protect after installation, but Android might automatically disable the app's services later.

About This Project

Z Flow is a Flutter-based automation tool developed for advanced Android users and developers. It solves the frustration of constant ad interruptions by monitoring system UI events and triggering native click actions.

Key Features & Logic:

  • Automated Ad Interception: Uses a node-tree traversal algorithm to find "Skip Ad" components and perform ACTION_CLICK.
  • AMOLED Saver Overlay Under Development: A dark overlay layer that covers the screen with pure black pixels. While it doesn't physically "turn off" the hardware display via system APIs, it mimics the appearance on AMOLED screens and locks touch input to allow background-like playback.
  • Intelligent Package Filtering: Logic triggers only when specific package names (like YouTube) are in the foreground, reducing CPU overhead.
  • Flutter + Kotlin Bridge: UI is built in Flutter, while system-level interaction is managed via a dedicated Kotlin-based Accessibility Service.

Technology Stack & Implementation

The app combines Flutter's rapid UI development with native Android capabilities for low-level system interaction.

  • Engine: Flutter (Dart)
  • Platform Bridge: Kotlin MethodChannels for native service management.
  • 核心 API: Android AccessibilityService for UI tree traversal and node-based interaction.
  • Overlay System: SYSTEM_ALERT_WINDOW permission for the AMOLED Saver overlay.
  • Optimization: Event-driven architecture ensures zero performance impact when video apps are not active.
