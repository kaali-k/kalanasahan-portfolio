About This Project

"K.S.D Finance" is a secure, private, and feature-rich mobile application designed for systematic and effortless personal finance management. Featuring a clean, user-friendly interface, the app empowers users to maintain complete control over their financial data, which is stored exclusively on their own device.

Key Features:

  • Comprehensive Financial Tracking: Easily record all income and expenses with customizable categories, detailed descriptions, and notes.
  • Insightful Financial Reports: Generate detailed PDF reports for any date range, visualizing total income, expenses, and net balance to understand financial health at a glance.
  • Advanced Security Options: Enable an additional layer of security with a PIN or Biometric App Lock to protect sensitive financial data, on top of secure user accounts.
  • Robust Data Backup & Restore: A powerful Export and Import feature allows you to back up and restore your entire financial history, ensuring a seamless transition between devices.
  • Personalized Experience: Switch effortlessly between a carefully designed Light Mode and a comfortable Dark Mode, and set timely reminders for important financial events.
  • Intuitive Data Management: Easily view, edit, or delete any record with simple swipe gestures, controlled via a centralized and user-friendly settings screen.

Project Philosophy & Development Focus:

This project demonstrates the end-to-end development of a complete, production-ready mobile application with a strong emphasis on modern principles:

  • Data Privacy by Design: The core architecture ensures all user data is stored locally. No financial information ever leaves the device unless explicitly exported by the user, guaranteeing 100% ownership and privacy.
  • User-Centric Security: Acknowledging the sensitive nature of financial data, multiple layers of security—from account passwords to biometric locks—were implemented to give users peace of mind.
  • Modular & Maintainable Code: The project is structured with a clean, modular architecture, utilizing custom hooks (e.g., `usePdfGenerator`) and a clear separation of concerns (UI, logic, database), ensuring the application is both scalable and easy to maintain.
  • Seamless User Experience (UX): Every feature was designed to be intuitive, responsive, and efficient, making personal finance management a pleasant and frictionless task.

Technology Stack

This application is engineered with a modern, performance-oriented technology stack, ensuring a secure, scalable, and truly native cross-platform experience.

  • Core Framework: React Native forms the backbone, allowing for a single, maintainable codebase that delivers native performance on both Android and iOS.
  • Local Database: SQLite (via `react-native-sqlite-storage`) is utilized for robust, persistent, and offline-first on-device data storage, guaranteeing user data privacy and control.
  • State Management & UI:
    • Navigation: A seamless and logical multi-screen user flow is orchestrated using React Navigation (`@react-navigation/native-stack`).
    • Dynamic Theming: A custom-built theming system using React's Context API and AsyncStorage allows users to instantly switch between Light and Dark modes.
    • Advanced Gestures: `react-native-gesture-handler` powers sophisticated touch interactions like the intuitive swipe-to-edit/delete functionality.
  • Native Modules & APIs:
    • Enhanced Security: Multi-layered security is implemented with an App Lock feature, supporting both PIN and Biometric Authentication (Fingerprint/Face ID) via the `react-native-biometrics` library.
    • Data Portability: A powerful backup and restore solution allows users to export their data as a JSON string using the native Share API and seamlessly import it.
    • PDF Generation & Handling: Financial reports are generated using `react-native-html-to-pdf`, enhanced with download completion push notifications (`react-native-push-notification`) and the ability to open files directly (`react-native-file-viewer`).
    • User Feedback: Clear and non-intrusive user feedback is provided through "toast" notifications using `react-native-toast-message`.
